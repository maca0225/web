CSS Positioning

The position property specifies the type of positioning method used for an element. There are 4 different position values: static, relative, fixed or absolute.

static

default value. elements render in order, as they appear in the normal document flow.

relative

the element is positioned relative to its normal position

fixed

the element is positioned fixed to the browser window

absolute

the element is positioned relative to its first positioned (non-static) ancestor element
